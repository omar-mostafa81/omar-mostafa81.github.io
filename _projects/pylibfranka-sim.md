**Pylibfranka Sim** is a MuJoCo simulation twin for the `pylibfranka` Python package, ensuring API compatibility
with `libfranka` for seamless real-to-sim transitions.

**Implemented controllers:**
- Joint Impedance Controller
- Cartesian Task-Space Impedance Controller
- Force Controller
- Force-Position Controller

Developed at the **Robot Learning & Control Lab, NYU Abu Dhabi** under Prof. Fares Abu-Dakka.
