**D1Py** provides a clean Python interface for the Unitree D1 robotic arm, built on top of `unitree_sdk2`. It abstracts
the low-level communication protocol and exposes a straightforward API for joint control, making the arm accessible for
research applications.

Developed during the internship at the **Machines in Motion Lab, NYU Tandon**.
